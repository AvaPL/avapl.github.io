# avapl.github.io

My personal website.

## Testing locally

(Optional) Initialize `devbox` shell which will install all required tools:

```shell
devbox shell
```

Install dependencies:

```shell
bundle
```

Run local server:

```shell
bundle exec jekyll s
```
